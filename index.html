O site "Contador Fibonacci" para que o usuário possa agora inserir a quantidade de termos que deseja ver na sequência de Fibonacci, tornando a experiência mais personalizada. O "Contador Fibonacci" é uma ferramenta interativa que calcula e exibe os números da sequência de Fibonacci, começando com 0 e 1. A sequência é gerada somando os dois últimos números para obter o próximo termo (por exemplo, 0 + 1 = 1, 1 + 1 = 2, e assim por diante). No site, o usuário insere o número de termos desejado, e a sequência correspondente é gerada automaticamente e exibida de forma clara. Isso facilita a exploração da sequência, seja para fins educacionais, curiosidade matemática, ou prática de programação. A interface simples, com um campo de entrada e um botão, torna a interação intuitiva. O código JavaScript foi ajustado para garantir que o site funcione de maneira eficiente e responsiva.
